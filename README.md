# Quantifying Antiquity — Audits

Constructability audits of mainstream ancient-construction claims.

Each audit takes the published numbers — their papers, their records, their
consensus — and runs the material-and-supply check any real construction
project gets before the first block moves. No alternative authorship. No alien
intervention. No lost civilizations. Just a falsifiable arithmetic test of what
a published method needs against what the documented record supplies.

**Channel:** [Quantifying Antiquity on YouTube](https://www.youtube.com/@QuantifyingAntiquity)

## How to read these

Every input is flagged as sourced, derived, or assumption, and all math is
shown. The headline figures are the anchored, conservative numbers; sensitivity
ranges and ceiling cases are labeled as such. Each document lists its sources.

## Found an error?

The math is on the table on purpose. If the arithmetic is wrong, the document
is wrong — that's why it's public. [Open an issue](../../issues) and show where.
That's how this gets better.
